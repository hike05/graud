# Changelog

All notable changes to Graud will be documented in this file.

## [1.0.0] - 2025-11-26

### Added
- Initial release
- Automated installation script with interactive prompts
- Draug (custom Caddy build) integration
- CrowdSec threat protection with auto-generated bouncer keys
- Automatic HTTPS with Let's Encrypt
- HTTP/3 support
- Health checks for all services
- Comprehensive documentation
- Production-ready configuration

### Features
- One-command installation
- Auto-generated CrowdSec bouncer API keys
- Rate limiting (100 requests/minute per IP)
- JSON access logging
- TLS 1.2/1.3 support
- Automatic HTTP to HTTPS redirect
- Docker Compose orchestration
- Forward proxy with basic authentication
